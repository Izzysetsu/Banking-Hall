# Banking Hall Display System

Sistem digital signage untuk menampilkan gambar dan video di TV/Monitor dengan pengaturan animasi dan durasi fleksibel.

## Fitur
- **Dashboard Admin**: Upload, atur urutan, atur animasi, atur durasi.
- **Slideshow Tampilan**: Memutar gambar & video secara loop tanpa terputus.
- **Support Hybrid Storage**:
  - **Lokal**: Menggunakan SQLite & folder `static/uploads/`.
  - **Cloud (Vercel Ready)**: Menggunakan Supabase Database & Supabase Storage.

---

## ⚡ Deploy Gratis ke Vercel + Supabase (Rekomendasi Hosting Gratis)

Agar aplikasi dapat di-hosting gratis di **Vercel** tanpa masalah data hilang (ephemeral storage):

### Langkah 1: Setup Supabase
1. Daftar/Login ke [supabase.com](https://supabase.com) dan buat proyek baru.
2. **Setup Database Table**:
   Di Supabase **SQL Editor**, jalankan skrip berikut:
   ```sql
   CREATE TABLE IF NOT EXISTS playlist (
       id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
       filename TEXT NOT NULL,
       type TEXT NOT NULL,
       url TEXT,
       duration INT DEFAULT 10,
       animation TEXT DEFAULT 'fade',
       order_index INT DEFAULT 0,
       created_at TIMESTAMPTZ DEFAULT NOW()
   );
   ```
3. **Setup Storage Bucket**:
   - Ke menu **Storage** -> **Create Bucket**.
   - Beri nama: `playlist-media`.
   - Centang **Public Bucket** (wajib agar gambar/video bisa diakses public).

4. **Ambil API Credentials**:
   - Buka **Project Settings** -> **API**.
   - Catat `Project URL` dan `anon / public key` (atau `service_role key`).

---

### Langkah 2: Deploy ke Vercel
1. Push kode aplikasi ini ke GitHub Anda.
2. Login ke [vercel.com](https://vercel.com) dengan akun GitHub.
3. Import repositori proyek ini.
4. Pada bagian **Environment Variables** di Vercel, tambahkan 3 variable berikut:
   - `SUPABASE_URL`: (Project URL Supabase Anda)
   - `SUPABASE_KEY`: (API Key Supabase Anda)
   - `SUPABASE_BUCKET`: `playlist-media`
5. Klik **Deploy**! Aplikasi Anda langsung aktif online secara gratis.

## Cara Menjalankan di Komputer Lokal (Uji Coba)

1. Pastikan Anda sudah menginstal [Python 3](https://www.python.org/).
2. Buka terminal/command prompt di folder aplikasi ini.
3. Jalankan perintah instalasi modul:
   ```bash
   pip install -r requirements.txt
   ```
4. (Opsional) Salin file `.env.example` menjadi `.env` jika ingin mencoba Supabase lokal.
5. Jalankan aplikasi:
   ```bash
   python app.py
   ```
6. Akses di browser:
   - Layar Tampilan (TV): `http://localhost:5000`
   - Dashboard Admin: `http://localhost:5000/admin`

## 🚀 Panduan Deploy ke VPS (Alternatif Production)

Saat Anda menyewa VPS (misalnya DigitalOcean, Linode, Hostinger, dll) dengan OS Ubuntu, ikuti langkah ini:

1. **Upload File ke Server**
   Gunakan FileZilla atau SCP untuk mengunggah seluruh folder ini ke server (contoh: di `/var/www/banking-hall`).

2. **Akses Server Melalui SSH**
   ```bash
   ssh root@IP_VPS_ANDA
   ```

3. **Install Python dan Gunicorn**
   ```bash
   sudo apt update
   sudo apt install python3 python3-pip python3-venv
   cd /var/www/banking-hall
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   pip install gunicorn
   ```

4. **Buat Service Systemd**
   ```bash
   sudo nano /etc/systemd/system/banking.service
   ```
   Isi dengan:
   ```ini
   [Unit]
   Description=Gunicorn instance to serve Banking Hall App
   After=network.target

   [Service]
   User=root
   Group=www-data
   WorkingDirectory=/var/www/banking-hall
   Environment="PATH=/var/www/banking-hall/venv/bin"
   ExecStart=/var/www/banking-hall/venv/bin/gunicorn --workers 3 --bind 127.0.0.1:5000 app:app

   [Install]
   WantedBy=multi-user.target
   ```
   Simpan dan aktifkan:
   ```bash
   sudo systemctl start banking
   sudo systemctl enable banking
   ```

5. **Install Nginx sebagai Reverse Proxy**
   ```bash
   sudo apt install nginx
   sudo nano /etc/nginx/sites-available/banking
   ```
   Isi dengan konfigurasi Nginx:
   ```nginx
   server {
       listen 80;
       server_name domainanda.com atau_IP_VPS_ANDA;

       location / {
           proxy_pass http://127.0.0.1:5000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
           proxy_set_header X-Forwarded-Proto $scheme;
           client_max_body_size 500M;
       }
   }
   ```
   Aktifkan Nginx:
   ```bash
   sudo ln -s /etc/nginx/sites-available/banking /etc/nginx/sites-enabled
   sudo systemctl restart nginx
   ```
